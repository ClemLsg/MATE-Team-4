## MODUP GIT

 - **Vérifier les fichier modifiés :**
  `git diff`

 - **Ajouter les fichier modifiés :**
  `git add *` ou  `git add *Nom du fichier*` 
  
 - **Vérifier les fichier ajoutés au commit:**
  `git status`

 - **Création du commit:**
  `git commit -m "Description"`

 - **Push du commit:**
  `git push`
